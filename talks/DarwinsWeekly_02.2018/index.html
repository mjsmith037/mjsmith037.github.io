A motivating example

This project stemmed from a statistical observation that arises when one is dealing with data which are strongly influenced by an underlying, one-dimensional gradient. Think, for example, of species distributions on an elevational gradient. For each species, there will be an elevation with the greatest abundance, and then ***basic assumption: nearby sites have similar species compositions

What is known

  • Ecologists have been interested in ordination for a long time
  • Fairly early on, Principal Component Analysis (PCA) was identified as a powerful, agnostic tool in this effort
  • For a period of time, the arch observed in PCA of one dimensional data was viewed as a statistical artifact
  • Now, the mathematics uderlying the arch are better understood and the emphasis has shifted away from "correcting" the ordination coordinates...
  • but the presence of the arch still presents challenges for extracting the optimal ordering of the data

# normalize the columns of the occurance matrix to get matrix B
M <- t(B) %*% B
# initialize total distance (smaller is better)
total_score <- 0
# loop through the columns of M to get each contribution to the total score
for (each column_i in M) {
  for (each column_j in M) {
    distance_i[j] <- sqrt(sum((column_i - column_j)^2))
  }
  rank_distance_i <- rank(distance_i)
  for (each column_j in M within d steps of column_i) {
    total_score <- total_score + sum(rank_distance_i[j])
  }
}
